Konsep Nilai Mutlak

Nilai mutlak suatu bilangan real x, ditulis |x|, menyatakan jarak bilangan tersebut dari titik nol pada garis bilangan. Karena jarak tidak pernah bernilai negatif, berlaku |x| ≥ 0 untuk setiap x.

Definisi Formal
|x| = x jika x ≥ 0
|x| = −x jika x < 0

Menjelaskan bahwa nilai mutlak selalu menghasilkan bilangan positif atau nol. Contoh: |−7| = 7, |5| = 5.

Fungsi dalam Matematika
  • Menghitung jarak dua titik pada garis bilangan: |a − b|
  • Menyelesaikan persamaan nilai mutlak kompleks
  • Mengukur galat (error) atau selisih dalam analisis statistik
Sifat-Sifat Penting
  • |xy| = |x| · |y| — Sifat Perkalian
  • |x/y| = |x|/|y| — Sifat Pembagian (y ≠ 0)
  • |x| = |−x| — Sifat Simetri Tanda Bilangan
